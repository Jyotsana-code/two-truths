# Two Truths

🌐 **Live Demo:** https://two-truths.netlify.app/

## About the Project
Two Truths is a prevention-first platform that verifies job opportunities and identifies trafficking risks before migration occurs.

It helps users:
- Verify job offer authenticity
- Detect scam and trafficking red flags
- Get an instant AI risk score
- Receive safety recommendations
